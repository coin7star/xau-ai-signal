# Step 10M — Mobile Header Polish

Patch ini merapikan header dashboard di tampilan HP.

Perubahan utama:
- status chip disusun grid 2 kolom di mobile
- badge feed dan status alert dibuat full width
- badge role, akses, dan tombol logout lebih rapi
- label role dibuat lebih ramah user (`Admin`, `Premium`, `Free`)
- label admin access diganti menjadi `Akses Admin`
