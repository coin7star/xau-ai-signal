# Step 10J Patch — Premium Feed Label

Patch kecil untuk membuat status bar dashboard terlihat lebih profesional.

## Perubahan

- Mengganti label teknis `Source: Firebase RTDB` menjadi `Market Feed: Premium Live Engine`.
- Mengganti `M1 data` dan `M15 data` menjadi stream candle yang lebih natural.
- Mengganti `Last close` menjadi `Last Price`.
- Mengganti `Data: Active/Waiting` menjadi `Feed Status: Online/Syncing`.

## Catatan

Patch ini hanya mengubah tampilan tulisan dashboard.
Tidak mengubah Firebase, API, MQ5, cron, Telegram, atau strategi signal.
