# Step 10AM — Premium User Friendly Copy

Patch ini merapikan bahasa website untuk user premium agar tidak terlihat seperti dashboard developer/admin.

Perubahan utama:
- Istilah teknis di area user premium diganti menjadi bahasa produk yang lebih ramah.
- Panel Telegram user dibuat lebih jelas: Sinyal Utama, Update Hasil, Cek Status, Putuskan.
- Tab History/Performance dibuat lebih user-friendly.
- Tab SMC AI menjelaskan uji live tanpa istilah internal berlebihan.
- Landing page dan paywall mengganti kata admin menjadi support/tim XAU AI di area user.

Catatan:
- Tidak mengubah logic sinyal.
- Tidak mengubah cron.
- Tidak mengubah Telegram delivery.
- Tidak mengubah Strategy A, M1 Scalp, atau SMC AI engine.
