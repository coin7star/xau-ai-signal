# Step 10Y1 — Broadcast Panel Polish

Upgrade panel Broadcast Telegram manual.

Perubahan:
- Estimated receiver berdasarkan target broadcast.
- Preview pesan sebelum dikirim.
- Counter karakter maksimal 900.
- Catatan perbedaan Manual Broadcast dan Auto Signal Alert.
- Cooldown server anti-spam via `TELEGRAM_BROADCAST_COOLDOWN_SEC` default 60 detik.
- Log broadcast tetap disimpan ke Firebase.

Catatan: fitur ini tidak mengaktifkan auto multi-user signal alert. Itu tetap disiapkan untuk Step 11.
